# HF Example Repo

This is a simple example project using Hugging Face transformers and nnsight and extract layers

## Setup

This project uses `uv` for Python dependency management and virtual environments.

### Prerequisites

- uv (https://github.com/indygreg/uv)

### How to run

To install dependencies:

```bash
uv sync
```

To run the project:

```bash
uv run python main.py
```
